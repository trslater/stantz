# Stantz

> What? What just popped in there?

Named after the "heart of the Ghostbusters," Ray Stantz, **Stantz** is a path tracer.

## Requirements

- Eigen3
- SDL2

## Installation

Just copy the files.

## Building

```
$ make
```

## Usage

```
$ ./bin/test <width> <height>
```

`width` and `height` are the output image size in pixels.
